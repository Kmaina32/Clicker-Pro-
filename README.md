# 🟢 CLICKER_PRO // v2.0

> **KERNEL_LEVEL_SIMULATION // HARDWARE_GRADE_AUTOMATION**

A high-performance, web-based auto-clicker simulator and script generator designed for precision automation. Built with a terminal-inspired "coder" aesthetic, **CLICKER_PRO** provides professional-grade control over click intervals, targeting, and execution cycles.

---

## ⚡ CORE_FEATURES

- **[ CPM_CONTROL ]**: Set precise Clicks Per Minute (CPM) for human-like rhythmic automation.
- **[ AUTO_DETECT ]**: Move your mouse to a target and click to automatically lock in X/Y coordinates.
- **[ RANDOM_JITTER ]**: Enable randomized intervals to bypass simple anti-cheat/detection systems.
- **[ STOP_MODES ]**: 
  - **Click Limit**: Stop automatically after a specific count.
  - **Duration Limit**: Stop automatically after a set time (seconds).
- **[ PRE_EXEC_COUNTDOWN ]**: Configurable delay before simulation begins.
- **[ SCRIPT_EXPORT ]**: Generate hardware-grade Python (`pyautogui`) scripts for local execution.
- **[ CPS_TESTER ]**: Integrated performance tester to measure your manual clicking speed.

## 🛠️ INSTALLATION_INIT

### Prerequisites
- [Node.js](https://nodejs.org/) (v18+)
- [npm](https://www.npmjs.com/)

### Setup
```bash
# Clone the repository
git clone <repository-url>

# Navigate to the project directory
cd clicker-pro

# Install dependencies
npm install

# Start the development server
npm run dev
```

## 📜 USAGE_PROTOCOL

1. **Configure**: Set your interval, unit (ms, s, m, cpm), and click type (left, right, double).
2. **Target**: Use `AUTO_DETECT` to lock in coordinates or stick to `MOUSE_POS`.
3. **Safety**: Set a `Click Limit` or `Duration Limit` to prevent infinite loops.
4. **Execute**: Hit `EXECUTE_SIMULATION` and wait for the countdown.
5. **Export**: Switch to the `Export Scripts` tab to get a Python version for local use.
